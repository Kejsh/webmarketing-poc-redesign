## 3) Kratki sadrzajni snapshot kljucnih stranica

- Home (`/`): hero, trust/logowall, rjesenja, EasyEdit diferencijator, proces, case studies preview, insights preview, zavrsni CTA.
- Work (`/work`): listing studija slucaja s filterima i CTA blokovima.
- EasyEdit (`/easyedit`): platformski pregled, pillar sekcije, FAQ, demo CTA.
- O nama (`/o-nama`): timeline tvrtke, principi, tim, trust brojke, CTA.
- Insights (`/insights`): lista clanaka, istaknuti clanak, resursi/guide blokovi.
- Kontakt (`/kontakt`): dual-path kontakt forma, kontakt podaci, trust mini i CTA.
